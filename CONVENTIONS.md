# 项目约定 — pdf-replace

> 此文件保存项目的协作约定，以便更换 AI 助手时也能遵守同一套规则。

## 1. 对话日志

- 每次对话内容追加保存到 `CONVERSATION.md`
- 用户发言前缀：`Yau:`
- AI 发言前缀：`Hermes:`（根据使用的 agent 名称）
- 只记录用户输入和 AI 回复，不记录工具调用日志
- 每次对话起始处标记日期时间

## 2. 代码修改提交

- 每次修改代码后，需执行以下流程：
  1. 运行测试 / 构建（确保代码通过）
  2. Git commit（提交信息用中文，简明描述改动内容）
  3. Git push（自动推送到远程仓库）

## 3. 其它

- 工作区有脏状态时，修改前自动 stash，修改提交后 pop 恢复
